## 🃏 **Super Trunfo - Países** 🌍

### 🚀 **Como Rodar** 💻

* **Passo 1:** 📄 Copie o código e salve em um arquivo chamado `super-trunfo-c.c`.

* **Passo 2:** 🖥️ Abra o seu terminal ou prompt de comando.

* **Passo 3:** 🛠️ Compile o programa usando o **GCC** com o comando:

`gcc -o super-trunfo-c super-trunfo-c.c`.

* **Passo 4:** ▶️ Execute o sistema digitando:

`./super-trunfo-c` (no Linux/Mac) ou `super-trunfo-c.exe` (no Windows).

---

### 🕹️ **Regras do Jogo** 📝

* 
**Cadastro:** ✍️ Insira os dados de duas cidades diferentes, como População, Área e PIB.

* **Cálculos:** 🔢 O sistema calcula automaticamente a **Densidade Populacional** e o **PIB per Capita**.

* **Batalha:** ⚔️ As cartas são comparadas atributo por atributo.

* **Vencedor:** 🏆 Para a **Densidade**, o menor valor vence; para os outros itens, ganha o maior valor.

* **Resultado:** ✅ O programa exibe `1` se a Carta 1 vencer e `0` se a Carta 2 vencer.

---

### 🛠️ **Requisitos** ⚙️

* **Compilador:** 🛠️ GCC ou uma IDE como Code::Blocks ou VS Code.

* **Linguagem:** 🔵 C padrão.
